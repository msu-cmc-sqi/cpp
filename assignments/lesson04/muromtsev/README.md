# Ментор по программированию

## Идея
Реализовать ИИ-агента, который будет отвечать на вопросы, анализировать код, предлагать его правки, а также будет отслеживать прогресс пользователя и стимулировать его развиваться дальше.

## Возможности
- Быстро и по сути отвечает на вопросы по программированию
- По запросу пишет код и объясняет синтаксис
- Производит отладку уже написанного кода
- Консультирует, какие участки кода можно доработать в уже написанной программе
- Напоминает, в чём пользователь со временем стал лучше
- Указывает пользователю на его постоянные ошибки и обращает внимание на его сильные стороны, как программиста


## Установка и сборка
```bash
chmod +x build.sh
chmod +x rebuild.sh
./build.sh
```
## Пересборка проекта
```bash
./rebuild.sh
```
## Запуск ии-агента
```bash
./run.sh <путь до конфига>
```
