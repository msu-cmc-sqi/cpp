## 6. JSON –±–∏–±–ª–æ—Ç–µ–∫–∞

### 6.1. **–°–æ–∑–¥–∞–Ω–∏–µ JSON-–æ–±—ä–µ–∫—Ç–æ–≤**

`nlohmann/json` –ø–æ–∑–≤–æ–ª—è–µ—Ç –ª–µ–≥–∫–æ —Å–æ–∑–¥–∞–≤–∞—Ç—å JSON-–æ–±—ä–µ–∫—Ç—ã —Å –ø–æ–º–æ—â—å—é —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–∞, —Å—Ö–æ–∂–µ–≥–æ —Å –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏–µ–π —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤.

#### –ü—Ä–∏–º–µ—Ä —Å–æ–∑–¥–∞–Ω–∏—è JSON-–æ–±—ä–µ–∫—Ç–∞:

```cpp

    // –°–æ–∑–¥–∞–Ω–∏–µ JSON-–æ–±—ä–µ–∫—Ç–∞
    json j = {
        {"name", "Alice"},
        {"age", 25},
        {"is_active", true},
        {"skills", {"C++", "Python", "Java"}}
    };

    std::cout << j.dump(4) << std::endl;  // –í—ã–≤–æ–¥ JSON —Å –æ—Ç—Å—Ç—É–ø–∞–º–∏
    return 0;
}
```

**–†–µ–∑—É–ª—å—Ç–∞—Ç:**
```json
{
    "age": 25,
    "is_active": true,
    "name": "Alice",
    "skills": [
        "C++",
        "Python",
        "Java"
    ]
}
```

#### **–ú–µ—Ç–æ–¥—ã –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å JSON-–æ–±—ä–µ–∫—Ç–∞–º–∏:**

- **`j["key"]`**: –î–æ—Å—Ç—É–ø –∫ –∑–Ω–∞—á–µ–Ω–∏—é –ø–æ –∫–ª—é—á—É.
- **`j.at("key")`**: –ê–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω—ã–π –±–µ–∑–æ–ø–∞—Å–Ω—ã–π —Å–ø–æ—Å–æ–± –¥–æ—Å—Ç—É–ø–∞ (–≤—ã–±—Ä–∞—Å—ã–≤–∞–µ—Ç –∏—Å–∫–ª—é—á–µ–Ω–∏–µ –ø—Ä–∏ –æ—Ç—Å—É—Ç—Å—Ç–≤–∏–∏ –∫–ª—é—á–∞).
- **`j.size()`**: –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ä–∞–∑–º–µ—Ä JSON-–æ–±—ä–µ–∫—Ç–∞ –∏–ª–∏ –º–∞—Å—Å–∏–≤–∞.
- **`j.contains("key")`**: –ü—Ä–æ–≤–µ—Ä—è–µ—Ç –Ω–∞–ª–∏—á–∏–µ –∫–ª—é—á–∞ –≤ –æ–±—ä–µ–∫—Ç–µ.
- **`j.dump()`**: –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –æ–±—ä–µ–∫—Ç JSON –≤ —Å—Ç—Ä–æ–∫—É. –ú–æ–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å –æ—Ç—Å—Ç—É–ø—ã –¥–ª—è –±–æ–ª–µ–µ —á–∏—Ç–∞–±–µ–ª—å–Ω–æ–≥–æ –≤—ã–≤–æ–¥–∞, –Ω–∞–ø—Ä–∏–º–µ—Ä `j.dump(4)`.
