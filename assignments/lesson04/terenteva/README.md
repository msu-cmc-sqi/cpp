# AI Music Recommendation Agent

## Идея
Музыкальный ИИ ассистент - ассистент, который анализирует эмоциональное состояние пользователя и подбирает персонализированные музыкальные композиции для психологической поддержки и настроения. 

## Возможности
- Анализ эмоционального состояния человека
- Интеллектуальный подбор музыки
- Персонализация рекомендаций
- Работа с API Яндекс.Музыка позволяет подбирать актуальные доступные треки
- Предоставляет структурированную информацию о каждом треке: альбом, жанр, длительность, ссылки для прослушивания
- Позволяет искать отдельные треки в Яндекс.Музыке

## Установка и сборка
Для сборки необходима установка библиотеки:
```
pip install yandex-music
```
Если данный способ не работает, можно попробовать альтернативный:
```
python3 -m venv music_venv
source music_venv/bin/activate
pip install yandex-music
```
Далее:
```
git clone ...
cd music-agent
mkdir build && cd build
cmake ..
make -j
```
Для работы ИИ-агента требуется токен сервиса Яндекс.Музыка. О том, как его получить, можно почитать здесь (есть много разных способов): https://yandex-music.readthedocs.io/en/main/token.html

Документация по библиотеке yandex-music: https://yandex-music.readthedocs.io/en/main/index.html
